1. install autopsy 3.1.1

2. Run autopsy, click Tools -> Python Plugin

3. copy Modules Folder to Python Plugin Folder

4. jython.jar overwrite to C:\Program Files\Autopsy-3.1.1\autopsy\modules\ext

5. vlfeat folder and model file copy to C:\Program Files\Autopsy-3.1.1

6. autopsy restart, create new case.

7. Double Click Data Sources

8. Right Click Data Source down item

9. Click Run Ingest Modules

10. Only Checked Image_Classifier

11. Start

12. input model file name

13. Result folder path is Desktop/Result
